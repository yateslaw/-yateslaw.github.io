单词拆分

Posted on 2024-04-02 In 算法 , 动态规划

题目链接

  • 0139. 单词拆分 - 力扣

题目大意

描述：给定一个非空字符串 sss 和一个包含非空单词的列表 wordDictwordDictwordDict 作为字典。

要求：判断是否可以利用字典中出现的单词拼接出 sss 。

说明：

  • 不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。
  • 1≤s.length≤3001 \le s.length \le 3001≤s.length≤300。
  • 1≤wordDict.length≤10001 \le wordDict.length \le 10001≤wordDict.length≤1000。
  • 1≤wordDict[i].length≤201 \le wordDict[i].length \le 201≤wordDict[i].length≤20。
  • sss 和 wordDict[i]wordDict[i]wordDict[i] 仅有小写英文字母组成。
  • wordDictwordDictwordDict 中的所有字符串互不相同。

解题思路

思路 1：动态规划

1. 划分阶段

按照单词结尾位置进行阶段划分。

2. 定义状态

sss 能否拆分为单词表的单词，可以分解为：

  • 前 iii 个字符构成的字符串，能否分解为单词。
  • 剩余字符串，能否分解为单词。

定义状态 dp[i]dp[i]dp[i] 表示：长度为 iii 的字符串 s[0:i]s[0: i]s[0:i] 能否拆分成单词，如果为 TrueTrueTrue 则表示可以拆分，如果为 FalseFalseFalse 则表示不能拆分。

3. 状态转移方程
  • 如果 s[0:j]s[0: j]s[0:j] 可以拆分为单词（即 dp[j]==Truedp[j] == Truedp[j]==True），并且字符串 s[j:i]s[j: i]s[j:i] 出现在字典中，则 dp[i] = True。
  • 如果 s[0:j]s[0: j]s[0:j] 不可以拆分为单词（即 dp[j]==Falsedp[j] == Falsedp[j]==False），或者字符串 s[j:i]s[j: i]s[j:i] 没有出现在字典中，则 dp[i] = False。
4. 初始条件
  • 长度为 000 的字符串 s[0:i]s[0: i]s[0:i] 可以拆分为单词，即 dp[0]=Truedp[0] = Truedp[0]=True。
5. 最终结果

根据我们之前定义的状态，dp[i]dp[i]dp[i] 表示：长度为 iii 的字符串 s[0:i]s[0: i]s[0:i] 能否拆分成单词。则最终结果为 dp[size]dp[size]dp[size]，sizesizesize 为字符串长度。

思路 1：代码解释

  1. 构建字典集合：首先，将给定的字典列表 wordDict 转换为一个哈希集合 wordDictSet，这样可以使得查找某个单词是否在字典中的操作更加高效。

  2. 动态规划数组：创建一个布尔类型的动态规划数组 dp，其中 dp[i] 表示字符串 s 的前 i 个字符是否可以被拆分成字典中的单词。

  3. 初始化：将动态规划数组的第一个元素 dp[0] 设置为 true，因为空字符串可以被认为是已经被拆分成功的。

  4. 动态规划：从字符串的第一个字符开始遍历到最后一个字符，对于每个位置 i，再从字符串的起始位置开始到位置 i，依次尝试将字符串拆分成两部分，其中一部分在之前已经被拆分成功，另一部分在字典中存在。如果满足这个条件，则将 dp[i] 更新为 true。

  5. 返回结果：最终返回动态规划数组的最后一个元素 dp[len(s)]，即整个字符串是否可以被拆分成字典中的单词。

思路 1：复杂度分析

  • 时间复杂度：O(n2)O(n^2)O(n​2​​)，其中 nnn 为字符串 sss 的长度。
  • 空间复杂度：O(n)O(n)O(n)。
